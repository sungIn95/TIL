## untracked => unmodified => modified => staged

예시)

1. 처음으로 만들면 => untracked(1단계)
2. 보고서 파일 add => Staged(4단계)
3. Staged를 commit하면 => unmodified(2단계)
4. 보고서 수정 modified(3단계)
